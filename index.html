Quant Platform

An easy to use gateway through which investors can easily invest into the fund. All investments will be executed via smart contracts, which can be studied at our github repositories. All investments will be accepted in form of Ether or/and Bitcoin based transactions and an equivalent amount of Hedger tokens will be given to the investor. The Hedger tokens are an ERC-20 protocol based tokens which will hold a value roughly equal to the NAV of the fund
